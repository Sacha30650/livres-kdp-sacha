## 📚 Collection 1 — « Au seuil des merveilles » (Apolline Verger)

### Statut
- **Cible finale** : 5 tomes (et pas 12 — décision actée le 13/05/26)
- **Tome 1 — « Lila et la clé du grenier »** : publié sur KDP, validé
- **Tome 2 — « Lila et le grenier des autres »** : publié, en attente validation Amazon
- **Tomes 3 à 5** : à produire

### Pourquoi 5 tomes et pas 12
- Évite le pattern KDP suspect "1 livre/semaine pendant 3 mois sur même collection IA"
- Évite la saturation TikTok (12 vidéos "nouveau tome de Lila" = lassitude)
- Permet de **diversifier le portefeuille** : 2 collections de 5 tomes rapportent plus qu'1 collection de 12
- Finir fort à 5 = mémorable et propre

### Cadence prévue Tomes 3-5
- Tome 3 : ~première semaine de juin 2026
- Tome 4 : ~mi-juin 2026
- Tome 5 : ~fin juin / début juillet 2026
- Espacement 10-15 jours entre publications

### Architecture narrative à choisir avant production T3
3 options proposées :
- **Option A — Les mondes du globe** : 1 tome = 1 culture (Inde, Brésil, etc.)
- **Option B — Les 4 éléments × N** : eau, feu, air, terre... + variations
- **Option C — Une menace s'éveille** : arc narratif global avec antagoniste fil rouge

→ **À trancher avant Tome 3**

## 📚 Collection 2 — Nouvelle collection « utile + plaisir » (à créer)

### Décision actée
- **Lancement après** finalisation Tome 5 de Lila (juillet 2026)
- **Périmètre interdit** : émotions et gestion émotionnelle (zone de la cousine sur Iggy)
- **Périmètre cible** : utile + narratif, audience parents qui cherchent un livre qui sert

### Pistes retenues à explorer (par ordre de préférence)

#### 🥇 Piste 1 — Premier savoir-faire / autonomie
- Concept : 1 personnage récurrent apprend une compétence du quotidien par tome
- Exemples de tomes : faire ses lacets, préparer son petit-déj, ranger sa chambre, prendre soin d'une plante, mettre la table, se brosser les dents, s'habiller selon le temps
- Tranche d'âge : 4-7 ans
- Format : album 24-32 pages, aquarelle
- Marché : énorme (achat parental motivé "mon enfant doit apprendre ça")
- Liberté narrative : haute (chaque tome est différent)
- TikTok-compatible : oui ("Aujourd'hui [perso] apprend à...")

#### 🥈 Piste 2 — Compétences sociales pratiques
- Concept : comment agir dans une situation sociale concrète
- Exemples : se faire un ami, résoudre un désaccord, dire non, demander de l'aide, partager, perdre un jeu sans pleurer
- Tranche d'âge : 4-8 ans
- Marché : très bon (post-Covid, fort intérêt parents/écoles)
- Différenciation Iggy : Iggy = soi vs soi (émotion interne) ; ici = soi vs les autres (action sociale)
- Risque : narrativement plus exigeant pour éviter le moralisateur

#### 🥉 Piste 3 — Curiosités naturalistes accessibles
- Concept : histoire narrative + mini-info véridique en fin de page
- Exemples : pourquoi les abeilles, comment poussent les arbres, d'où vient la pluie
- Marché : porteur (parents recherchent du slow content éducatif)

### Pistes écartées
- ❌ Émotions (Iggy / cousine)
- ❌ Princesses anti-conventions (saturé)
- ❌ Premier raisonnement mathématique (trop scolaire)
- ❌ Apprendre langue étrangère (workflow incompatible)

### Décisions à prendre avant lancement Collection 2
1. **Tranche d'âge cible** : 6-8 ans (continuité Lila) ou 3-6 ans (continuité Iggy, marché plus volumineux) ?
2. **Format** : album 32 pages (Lila) ou plus court 16-20 pages (production plus rapide, plus de tomes possibles) ?
3. **Nom de plume** : conserver Apolline Verger (= 2 collections sous le même nom) OU créer une 3e identité ?
4. **Nom de collection** : à trouver une fois la piste 1/2/3 choisie

## 🎯 Stratégie de distribution — Phases A / B / C

### Phase A — IngramSpark (référencement multi-canaux online)
- **Quand** : après validation de la collection sur 3-4 tomes (donc fin juin 2026 minimum pour Lila)
- **Coût** : ~50 $ par livre setup + ISBN à l'AFNIL (lot de 10 à 140 €)
- **Délai** : 6-8 semaines avant mise en circulation
- **Apport** : commandable sur fnac.com, cultura.com, decitre.fr, leslibraires.fr + via Ingram international
- **Pré-requis** : SIRET (donc déclaration AA validée)

### Phase B — Dilicom + démarchage librairies
- **Quand** : Q3-Q4 2026 quand 4-5 tomes par collection disponibles
- **Coût** : inscription Dilicom ~150-300 €/an + temps démarchage
- **Apport** : ouvre les commandes pros (médiathèques, écoles, librairies indépendantes via Dilicom)
- **Action démarchage** : 30 librairies indépendantes jeunesse ciblées dans 5 grandes villes
- **Argument fort** : levier TikTok 27k (proposer dédicaces + annonce aux abonnés en échange de mise en rayon)

### Phase C — Distributeur français
- **Quand** : 2027 au plus tôt, si Phase B confirme l'engouement
- **Coût** : marge revue à la baisse (le distributeur prend 50-60%)
- **Candidats** : Pollen (petit, jeunesse, accessible), Harmonia Mundi (moyen), Interforum (gros, exigeant)
- **Pré-requis** : 8-12 tomes au catalogue + ventes physiques régulières prouvables

## ⚠️ Risques identifiés et plans de mitigation

### Risque 1 — Bannissement KDP
- **Probabilité** : faible (~10-15%) si cadence respectée
- **Signaux d'alerte** : Content Quality Notice, déréférencement progressif, ventes qui chutent sans raison
- **Plan de mitigation** :
  - Cadence raisonnable : pas plus d'1 livre/semaine sur même compte
  - Diversifier les mots-clés et descriptions entre tomes
  - Commander Author Copies systématiquement (signal humain)
  - Si signal d'alerte : pause de 2 semaines + migration accélérée vers IngramSpark
- **Backup** : tout est archivé sur GitHub, donc en cas de bannissement on republie ailleurs

### Risque 2 — Saturation TikTok
- **Probabilité** : moyenne sur 12 publications rapprochées
- **Plan de mitigation** :
  - Varier le contenu (pas que "nouveau tome")
  - Espacer les annonces : 1 vidéo livre toutes les 2 semaines max
  - Cross-content : extraits lecture, coulisses, témoignages parents, etc.

### Risque 3 — Conflit avec la cousine sur Iggy
- **Plan de mitigation actuel** :
  - 100% séparation des collections (nom de plume, univers, méthode)
  - Système de reversement à mettre en place (voir conversation séparée "Gestion financière KDP")
  - Justificatifs PDF mensuels avec déduction Amazon Ads
- **Action** : initier le statut Artiste-Auteur en séparé, garder des comptes propres et traçables

### Risque 4 — Réception négative sur l'IA dans les illustrations
- **Constaté** : quelques reviews Amazon sur Iggy mentionnent l'IA négativement
- **Plan de mitigation** :
  - Déclaration "AI-generated content" rigoureuse sur KDP
  - Travail de cohérence des personnages (refs Higgsfield) pour éviter les défauts visibles
  - Vérification page par page systématique (cf. corrections T2 sur skulls, 6 enfants, etc.)
  - Author Copy avant publication pour valider rendu imprimé

## 🧰 Pipeline technique opérationnel

### Workflow actuel (validé)
1. **Production images** : Higgsfield Nano Banana Pro 2K avec image-ref personnage
2. **Téléchargement** : a-Shell iPhone → Python one-liner → push direct GitHub
3. **Assemblage PDF** : scripts Python (PIL) avec template format KDP-ready
4. **Validation visuelle** : preview HTML générée auto, vérif sur iPhone
5. **Push GitHub** : commit automatique, accès depuis n'importe où
6. **Upload KDP** : depuis lien GitHub vers KDP

### Temps moyen par tome
- Production (avec Claude) : ~1h
- Téléchargement images : ~5 min via a-Shell
- Génération PDFs : ~3 min
- Upload KDP : ~10 min
- **Total** : ~1h20

### Crédits Higgsfield par tome (moyenne)
- ~118 cr / tome
- Plan Ultimate : ~1141 cr/cycle
- Donc ~10 tomes possibles par cycle Higgsfield
